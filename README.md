## About me

I'm a software engineer who loves building products from scratch — not just writing code, but owning the whole thing end to end. Right now I'm building my own startup, **[Gamehok](https://gamehok.com)**, a consumer tech platform for gamers where I wear a lot of hats: backend, infra, AI, and whatever else needs doing.

Before this, I was an early/core engineer at an ed-tech startup where I built the features that powered the product’s growth — gamification systems with streaks and leaderboards, a learner analytics dashboard, video engagement tracking, and a versioning system for e-learning content with rollback support.

Some things I've shipped recently:
- 🧘 **NeuraBreak** — a fully offline desktop app that keeps desk workers healthy with real-time monitoring, eye break reminders, and smart break scheduling
- 📹 An **in-house video transcoding + adaptive streaming pipeline** for Gamehok (cut external dependencies significantly)
- 🎮 An **AI-powered leaderboard engine** that auto-generates fair rankings across game formats for Gamehok

I've always been a problem-solver at heart — back in college I was into competitive programming, hitting 1631 on HackerEarth, 1615 (3★) on CodeChef, and 1613 on LeetCode. These days that same thinking goes into designing systems.
